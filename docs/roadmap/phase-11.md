### Phase 11: Analytics & Gamification (Weeks 14-15)

**Goal:** Motivate users with insights and progression systems.

**Deliverable:** Stats dashboard with XP, levels, and achievements.

- [ ] **Stats Dashboard**

  - [ ] Create Stats screen with overview cards
  - [ ] Implement time distribution pie chart (by context)
  - [ ] Add planned vs. actual bar chart (daily/weekly)
  - [ ] Show adherence score with trend (7-day, 30-day)
  - [ ] Display total focus time and productivity metrics
  - [ ] Add exportable reports (PDF or image)

- [ ] **XP & Leveling System**

  - [ ] Define XP formula (base XP per task, multipliers)
  - [ ] Implement level progression (10 levels, exponential curve)
  - [ ] Create level-up animation (confetti, sound effect)
  - [ ] Show XP progress bar on home screen
  - [ ] Award bonus XP for focus sessions (2x)
  - [ ] Award bonus XP for streaks

- [ ] **Streaks**

  - [ ] Track "Focus Streak" (consecutive days >80% completion)
  - [ ] Show current streak prominently in UI
  - [ ] Display longest streak (personal best)
  - [ ] Add streak freeze (1 per week, protects from breaking)
  - [ ] Show streak calendar (contribution graph style)
  - [ ] Celebrate milestone streaks (7, 30, 100 days)

- [ ] **Achievement System**

  - [ ] Design 20+ achievements (Early Bird, Night Owl, Perfectionist, etc.)
  - [ ] Implement achievement tracking logic
  - [ ] Show achievement unlock animation
  - [ ] Create achievements gallery (locked/unlocked)
  - [ ] Add rare/hidden achievements (Easter eggs)

- [ ] **Unlockables**

  - [ ] Unlock app icon variants (5 icons, unlock at levels 2, 4, 6, 8, 10)
  - [ ] Unlock color themes (5 themes tied to achievements)
  - [ ] Add "Pro" theme at level 10
  - [ ] Create badge system for profile (if social features added later)

- [ ] **Charts & Visualizations**
  - [ ] Integrate Recharts library
  - [ ] Create responsive, interactive charts
  - [ ] Add animations to chart rendering
  - [ ] Support touch interactions (tooltips on tap)
  - [ ] Implement chart export (save as image)
