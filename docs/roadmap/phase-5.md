### Phase 5: Contexts & Visual Customization (Week 7)

**Goal:** Let users organize tasks with color-coded categories.

**Deliverable:** Context management system that visually distinguishes life areas.

- [ ] **Context CRUD**

  - [ ] Create Contexts management screen
  - [ ] Implement add/edit/delete for contexts
  - [ ] Add context form (name, color, icon)
  - [ ] Create 10 default contexts (Work, Health, Social, etc.)
  - [ ] Allow reordering contexts (drag to reorder)

- [ ] **Context Integration**

  - [ ] Add context selector to task/block forms
  - [ ] Apply context colors to timeline blocks (tinted glass)
  - [ ] Show context icon on blocks (optional toggle)
  - [ ] Create context filter on timeline (show/hide by context)
  - [ ] Add context legend on timeline screen

- [ ] **Theme System**

  - [ ] Implement light/dark/system theme toggle
  - [ ] Create 5 preset themes (Ocean, Sunset, Forest, Midnight, Rose)
  - [ ] Add custom accent color picker
  - [ ] Save theme preference in IndexedDB
  - [ ] Apply theme instantly with smooth transition

- [ ] **Visual Customization**
  - [ ] Allow per-task icon customization
  - [ ] Add per-task color override option
  - [ ] Create icon size preference (small, medium, large)
  - [ ] Add timeline density setting (compact, comfortable, spacious)
